# last-method

Adds a last method to all arrays. Will return -1 if the array is empty.

## Install
```javascript
npm i last-method
```

## Examples
```javascript
import 'last-method'

const array = [1, 2, 3, 4, 5]
array.last() // 5

const emptyArray = []
emptyArray.last() // -1
```
